# IBMChatBotSetup
Hands on Practical on Building a ChatBot using IBM Watson X Platform and access to IBM Cloud for Hands on Labs

<h2> Tools/Environments Used</h2>

- <b>IBM Build-Your-Own Chatbot Course</b> 
- <b>IBM Cloud Platform</b>
- <b>Watson X </b>

<h2>Project Activities performed & Images of its walk-through:</h2>


- <b>Designed and built an interactive AI-powered chatbot using IBM Watson Assistant on the IBM Cloud platform.</b>

- <b>Created and structured intents, entities, and dialog flows to enable the chatbot to understand and respond to predefined user queries effectively.</b>

- <b>Implemented a fallback mechanism to handle unrecognized questions gracefully, ensuring a consistent user experience.</b>

- <b>Deployed the chatbot and successfully integrated it into a WordPress website, enabling live user interactions and real-time testing.</b>
